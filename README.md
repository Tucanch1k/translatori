# Транслятор арифметических выражений с функцией pow

## Описание

Это учебная программа-транслятор, реализующая:

- **Лексический анализ (сканер)** — преобразует строку в поток токенов  
- **Рекурсивный нисходящий парсер с семантикой** — разбирает выражения согласно грамматике и вычисляет результат  
- **Семантический анализ** — встроен в парсер (вычисляет значение выражений)  
- **Обработка ошибок** — на уровне лексики и синтаксиса

Программа поддерживает арифметические выражения с операциями `+`, `*`, скобками и функцией `pow(a,b)`.

## Используемая грамматика

Expr -> Term
Expr -> Expr + Term
Term -> Factor
Term -> Term * Factor
Factor -> num
Factor -> pow_func
pow_func -> pow ( Expr , Expr )

### Токены
num - токен, задающий какое-либо число


## Файлы проекта

*scanner.py     # Лексический анализатор
*my_parser.py   # Рекурсивный нисходящий парсер с семантикой
*main.py        # Точка входа, пользовательский ввод


## Требования

* Python ≥ 3.6 (рекомендуется 3.8+)
* Модули: стандартная библиотека Python


## Запуск программы:

1. Откройте терминал в папке проекта
2. Выполните команду: python main.py
3. Введите арифметическое выражение с +, *, скобками и pow()


## Примеры использования:

pow(2, pow(3,4))
Результат: 2.4178516392292583e+24

2 + 3 * pow(2,3)
Результат: 26.0


## Обработка ошибок

* LexerError — недопустимые символы или неизвестные идентификаторы
* ParserError — нарушение грамматики, лишние символы после конца выражения
